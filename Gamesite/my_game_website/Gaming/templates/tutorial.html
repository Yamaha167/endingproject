Tutorial: How To Package and Upload Your Pygame Project

The project is set up to host Python games built with Pygame. To upload your work, you'll need to perform the following steps:

  1. Create a game using the Python Pygame library.
  2. Package the Pygame game with Pygbag.
  3. Create a thumbnail.
  4. Upload the files to the website.
  5. Enjoy!

1. Create a Game Using Python Pygame Library

Pygame is a set of Python modules designed for writing video games, adding functionality on top of the SDL library. This allows you to create fully featured games and multimedia programs in Python. For more information, visit Pygame Documentation.

2. Package the Pygame Game with Pygbag

Before starting, ensure you have a Pygame app that runs locally. Then, follow these steps:

  1. Install Pygbag using the terminal command: 
    pip install pygbag
  2. Rename your main file to 
    main.py
  3. Import asyncio in your main.py: 
    import asyncio
  4. Define your game loop within an async function: 
    import asyncio
import pygame
# import other necessary modules

async def main():
    while True:
        # Game loop
        pygame.display.update()
        await asyncio.sleep(0)

asyncio.run(main())
  5. Change to your project's parent directory and run Pygbag: 
    pygbag <project name>

After these steps, find the .apk and index.html files in: "project_folder_name"/build/web.

3. Create a Thumbnail

Create any PNG image to use as a thumbnail to represent your game on our website.

4. Upload the Files to the Website

After registering and logging in, navigate to the "Upload" page to submit your game and thumbnail.

5. Enjoy

If all steps are followed correctly, your game will be available on the "Games" page!
